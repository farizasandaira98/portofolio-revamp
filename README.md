# Svelte Portfolio

A modern, interactive portfolio website built with Svelte, Vite, and Tailwind CSS.

## Features
- Custom animated cursor
- Smooth scrolling navigation
- Responsive burger menu for mobile
- Intersection Observer for scroll animations
- Glass morphism and gradient text effects
- Timeline for experience and education
- Project showcase with tech stack tags
- Contact form with validation
- Language toggle (English/Indonesian)
- Dark/Light mode toggle
- UI auto-hide on scroll (with fade transition)

## Getting Started

### Prerequisites
- Node.js (v16+ recommended)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Folder Structure
- `src/lib/` — Svelte components and stores
- `src/app.css` — Tailwind and custom styles
- `public/` — Static assets

## Customization
- Edit translations in `src/lib/stores.js`
- Add projects in `src/lib/ProjectsSection.svelte`
- Update experience/education in their respective components
